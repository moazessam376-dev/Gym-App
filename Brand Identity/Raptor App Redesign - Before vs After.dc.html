How we get there

Four steps, in order of leverage.

01
Swap the theme tokens
Rewrite colors.ts, typography.ts, effects.ts. Because every screen reads @/theme, this reskins the whole app in one change — navy→onyx, blue→cyan, drop the glass/glow, Space Grotesk→Geist.
~1 day
02
Mono every number + kill the cringe
Add a mono text variant for all KPIs/stats/ranks. Sweep the i18n strings (en.json/ar.json) — replace "Workout crushed!" and 🔥 with factual lines.
~1 day
03
Build the tier system
The real feature work: tier computation (body-comp + adherence + goals), 8 crests, divisions, seasons, regional ranking. Bigger than a reskin — scope it as its own phase.
phase
04
Commission the crest artwork
The placeholders ship fine for v1; real illustrated crests are what make the competitive framing feel earned. Brief an illustrator off the design-system palette.
parallel
