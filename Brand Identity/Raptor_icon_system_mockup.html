04 · Specs

Drawing rules.

Stroke width
2px @ 24px native
Scales proportionally. At 16px use 1.5px. At 32px use 2.5px.
Line caps & joins
Round / Round
Always. Sharp caps break the system's softness.
Canvas
24×24 viewBox
2px padding on all sides. Content lives in 20×20 safe area.
Color · active
#3FD9C0 Signal cyan
Selected tab, active state, primary action.
Color · inactive
#7E828D Mist gray
Unselected, resting state.
Color · semantic
Cobalt / Purple / Green / Amber / Ember
Reserved for notification types. Never used in chrome.
